Favourite R pacakges: flexsurv

2019-10-28

I’m going to write about some of my favourite R packages. I’ll start with flexsurv https://github.com/chjackson/flexsurv-dev by Chris Jackson, which can be used to fit all kinds of parametric models to survival data. It can really do a lot, but I’ll pick out just 2 cool things:

  1. Fit a standard survival model, but where you don’t have to spend ages trying to work out what the parameters mean.
  2. Fit a proportional hazards model, which is a lot like a Cox model, but where you also model the baseline hazard using a spline.
